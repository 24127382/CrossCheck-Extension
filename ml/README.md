# ML Experiments

This folder contains ML experiments and research work.

## Structure

- **experiments/**: Experimental setups for different approaches
  - `fever_baseline/`: FEVER dataset baseline experiments
  - `entailment_eval/`: NLI model evaluation
  - `clip_similarity/`: CLIP-based similarity experiments

- **notebooks/**: Jupyter notebooks for exploration and analysis
- **evaluation/**: Evaluation metrics and benchmarking
- **datasets/**: Raw datasets (FEVER, evidence collections)
- **checkpoints/**: Saved model checkpoints
- **training/**: Training scripts and logs
