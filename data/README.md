# Data Directory

## Structure

- **raw/**: Raw datasets from external sources
  - FEVER dataset
  - Wikipedia dump
  - Evidence collections

- **processed/**: Processed and cleaned datasets
  - Formatted evidence documents
  - Tokenized claims
  - Embeddings cache

- **cache/**: Runtime caches
  - Embedding cache
  - Retrieval results cache
  - Model outputs cache

- **demo/**: Demo data for testing
  - Sample claims
  - Example evidence
  - Test datasets
